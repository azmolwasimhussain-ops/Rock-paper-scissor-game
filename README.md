# 🪨 Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors** game built using **React.js**.
Challenge the computer, track your score, and try to build the longest winning streak!

---

## 🚀 Features

* 🎮 Play Rock, Paper, Scissors against the computer
* 🤖 Random computer move generation
* 📊 Live score tracking (User vs Computer)
* 🔥 Streak & Maximum Streak tracking
* 🕹️ Round counter
* 🧠 Game result display (Win / Lose / Draw)
* 📜 Match history (previous moves)
* 🔄 Reset game functionality
* 🎨 Clean and modern UI with CSS styling

---

## 🛠️ Tech Stack

* **React.js** (Frontend)
* **JavaScript (ES6)**
* **CSS**
* HTML

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/rock-paper-scissors.git
```

2. Navigate to the project folder:

```bash
cd rock-paper-scissors
```

3. Install dependencies:

```bash
npm install
```

4. Run the app:

```bash
npm run dev
```

---

## 🎯 How to Play

* Click on any move:

  * 🪨 Rock
  * 📄 Paper
  * ✂️ Scissors
* The computer will randomly choose its move
* The result will be displayed instantly
* Scores, streaks, and history will update automatically
